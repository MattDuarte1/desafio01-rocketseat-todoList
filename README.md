# 💡 About

**Simple React To-Do List App. Made for Rocketseat's React Ignite Challenge.**

# ⚙️ How to run

### 📃 Dependencies

- **Typescript**
- **React**
- **Vite**
- **Phosphor-icons/react**

### On PC

- **Clone this repository**
- **Inside project's folder, run:**

```bash
$ npm install
```

- **And then, run:**

```bash
$ npm run dev
```

# 📲 Functionalities

- **You can create, check, remove and track your tasks**
